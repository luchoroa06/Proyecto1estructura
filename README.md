Es un sistema de análisis de redes de interacción de proteínas implementado como un grafo no dirigido utilizando listas de adyacencia, que permite a los usuarios cargar y guardar datos desde archivos CSV, visualizar gráficamente la estructura completa de la red, identificar las proteínas más conectadas (hubs), detectar complejos proteicos mediante algoritmos de búsqueda en anchura (BFS) y profundidad (DFS), y encontrar rutas metabólicas óptimas utilizando el algoritmo de Dijkstra.
